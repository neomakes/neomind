# 🧠 Personal Intelligence Platform

## "Palantir for Individual" - 개인 생활 지능화 플랫폼

**프로젝트명**: WienerMachine  
**작성일**: 2025-12-08  
**상태**: 기획 단계 (프로토타입 개발 준비)  
**비전**: 개인의 마음-행동-신체 데이터를 통합하여 생산성과 삶의질을 혁신적으로 개선하는 AI 솔루션

---

## 🎯 핵심 아이디어

기업의 의사결정을 지원하는 **Palantir Gotham/Foundry**처럼, 개인의 생활 데이터를 통합·분석하여 **개인화된 생산성 및 웰니스 인텔리전스**를 제공합니다.

```
Palantir (조직 수준): "조직은 이 전략을 취하세요"
                                ↓
                    Personal Intelligence
                                ↓
개인 솔루션 (개인 수준): "당신은 이 행동을 지금 해야 합니다"
```

```
✗ 데이터는 많지만 인사이트는 없다

  - "생산성이 낮아요" (O)

  - "운동하세요" (모든 사람 동일)
```

## ✅ 솔루션: 4계층 아키텍처

### Layer 1: Data Integration (데이터 통합)

**목표**: 이질적인 생활 데이터를 하나의 시간축으로 통합

#### 1.1 수집되는 데이터

```
┌─────────────────────────────────────────────────┐
│           개인 생활 데이터 (Multi-Modal)            │
├─────────────────────────────────────────────────┤
│                                                 │
│  [마음 (Mind)]      [행동 (Behavior)]  [신체 (Body)]
│  ─────────────      ──────────────    ─────────
│  • 기분              • 활동 패턴          • 심박수
│  • 스트레스           • 운동 강도          • HRV
│  • 집중력            • 일 집중도          • 수면
│  • 불안감            • 사교 활동          • 에너지
│  • 동기부여           • 시간 관리          • 피로도
│  • 창의성             • 회의 참석         • 스트레스 바이오마커
│  • 소진도             • 휴식 패턴         • 혈당 (선택)
│                     • 작업 전환         • 혈압 (선택)
│                                                 │
└─────────────────────────────────────────────────┘
```

#### 1.2 수집 방식

```yaml
마음 (Mind):
  빈도: 일 1~3회
  방식: EMA 설문 (아침/점심/저녁 1분)
  도구: 모바일 앱 푸시

행동 (Behavior):
  빈도: 실시간 (15분 단위)
  방식: 자동 추적 (스마트폰, 스마트워치, 캘린더)
  도구: 자동 데이터 수집

신체 (Body):
  빈도: 실시간~시간 단위
  방식: 웨어러블 연동 (Oura, Apple Watch)
  도구: API 연동

환경 (Context):
  빈도: 일 1회
  방식: 날씨, 계절, 특수 이벤트
  도구: 공개 API + 사용자 입력
```

#### 1.3 데이터 정규화

```
비동기 다중 빈도 데이터 → 공통 시간축으로 정렬

Mind: 일 3회 (평균/최댓값으로 일단위 변환)
Behavior: 15분 단위 (합계/피크 시간대로 일단위)
Body: 시간 단위 (평균/변동폭으로 일단위)

결과: 모든 데이터가 "일" 단위로 정렬
```

### Layer 2: Pattern Discovery (패턴 발견)

**목표**: 개인의 숨겨진 패턴 자동 발견

#### 2.1 멀티 모달 통합 학습

```
Mind Data  ─→  Mind Encoder (GRU, 32D)
                         ↓
Behavior Data ─→ Behavior Encoder (GRU, 64D) ← [현재 진행 중]
                         ↓
Body Data ─→ Body Encoder (GRU, 32D)
                         ↓
                Cross-Modal Attention
                (상호작용 학습)
                         ↓
            ┌────────────────────────────┐
            │ Shared Latent Space        │
            │ (공유 웰니스 표현)          │
            │                            │
            │ z_wellness: 전체 상태 (32D)│
            │ z_mind: 마음 특화 (16D)    │
            │ z_behavior: 행동 특화 (32D)│
            │ z_body: 신체 특화 (16D)    │
            └────────────────────────────┘
```

#### 2.2 발견되는 패턴

```
1. 시간적 패턴 (Temporal Patterns)
   예: "매주 월요일은 스트레스 높음"
       "금요일 저녁 생산성 급락"
       "월 4주차 피로도 최고조"

2. 상태 클러스터 (State Clusters)
   예: "High Energy + Low Stress + Good Sleep" ← 최적 상태
       "Burned Out" ← 위험 상태 (High Stress + Low Sleep)
       "Flow" ← 생산성 상태 (Deep Focus + High Productivity)

3. 전이 패턴 (Transition Patterns)
   예: "정상 → 위험" 전이 신호
       "부족 → 회복" 최적 시간대

4. 개인화된 최적 조건
   예: "당신의 생산성 피크: 화요일 09:00~11:00
        (HRV > 40, 수면 7.5시간 후)"
```

#### 2.3 이상 탐지 (Anomaly Detection)

```
조기 경고 신호:
├─ 번아웃 위험 (Stress ↑ + Sleep ↓ + Activity ↓)
├─ 우울증 위험 (Mood ↓ + Energy ↓ + Social ↓)
├─ 과훈련 (High Activity + Low Recovery)
└─ 질병 신호 (HRV ↓, Sleep Quality ↓, Fatigue ↑)

→ 실시간 감지 및 조기 경고
```

### Layer 3: Causal Inference (인과 추론)

**목표**: 상관관계에서 인과관계로 → 행동 가능하게

#### 3.1 상관 vs 인과

```
상관관계 (관찰만):
"수면이 적은 날 생산성이 낮다" ← 수동적

인과관계 (행동 가능):
"수면 부족이 생산성을 낮춘다" ← 능동적
→ "수면을 늘리면 생산성이 높아진다" (행동 가능)
```

#### 3.2 개인화된 인과관계 발견

```
예시 시나리오:

표준 조언: "운동하세요" (모든 사람 동일)

개인화된 인과:
- Person A: 운동 → 에너지 ↑ 45% (효과 있음)
- Person B: 운동 → 피로 ↑ 30% (역효과)
- Person C: 운동 → 스트레스 ↑ 25% (시간 부족)

권장:
- A: "매일 운동하세요"
- B: "운동 대신 요가/스트레칭"
- C: "20분 짧은 운동으로 시작"
```

#### 3.3 인과 강도 측정

```python
# 개인별 인과 강도 (Treatment Effect)

sleep_effect = 0.45  # 1시간 추가 수면 → 생산성 45% 향상
exercise_effect = 0.38  # 운동 → 오후 기분 38% 개선
stress_effect = -0.52  # 스트레스 ↑ → 수면의질 52% 악화

# 개인화된 효과 (이 사람만)
my_sleep_effect = 0.48  # 표준보다 조금 더 민감
my_exercise_effect = 0.42  # 표준보다 더 효과적
my_stress_effect = -0.58  # 표준보다 훨씬 민감
```

#### 3.4 개입 시뮬레이션

```
질문: "내가 2주간 매일 아침 30분 운동하면?"

예상 결과:
├─ 수면의질: 7.2 → 7.8 (↑0.6, 개선)
├─ 에너지: 6.1 → 7.4 (↑1.3, 큰 개선)
├─ 생산성: 6.5 → 7.6 (↑1.1, 개선)
├─ 스트레스: 5.2 → 4.1 (↓1.1, 개선)
└─ 확신도: 78%

→ "네, 효과 기대 가능"
```

### Layer 4: Recommendation (권장사항 생성)

**목표**: 행동 가능한 개인화된 권장사항 생성

#### 4.1 일일 권장사항 (매일 아침)

```
오늘의 인텔리전스:
═══════════════════════════════════════════

⚠️ 경고: 당신의 생산성이 위험합니다

현재 상태 분석:
├─ 어제 수면: 5.5시간 (평소 7.5시간, -2시간)
├─ 현재 스트레스: 8/10 (평소 5/10)
└─ 에너지: 4/10 (평소 7/10)

예상 오늘 생산성: 5/10 (위험 수준 ⚠️)

원인:
├─ 1순위: 수면 부족 (영향도 65%)
│  "수면은 당신의 생산성의 가장 강력한 드라이버"
│  증거: 당신의 패턴에서 1시간 수면 부족 → 생산성 45% 감소
│
└─ 2순위: 높은 스트레스 (영향도 35%)
   "스트레스 8 이상일 때 집중력 50% 감소"

───────────────────────────────────────────

오늘 해야 할 3가지 (우선순위):

1️⃣ [지금 해야할 것] 오전 30분 운동
   왜: 스트레스를 38% 줄일 수 있습니다
   어떻게: 산책 또는 가벼운 조깅
   기대 효과: 오전 생산성 +20%

2️⃣ [점심 후] 15분 명상
   왜: 당신의 점심 후 집중력 급락 해결 최고의 방법
   어떻게: Headspace (당신이 좋아하는 스타일)
   기대 효과: 오후 생산성 +15%

3️⃣ [저녁] 9시 정각 수면 시작
   왜: 충분한 수면이 내일을 결정합니다
   어떻게: 취침 1시간 전 화면 금지
   기대 효과: 내일 생산성 +35%

───────────────────────────────────────────

종합 아웃룩: ⚠️ 주의 필요
예상 오늘: 5/10 (위험)
개입 후: 6.5/10 (개선됨)
```

#### 4.2 주간 리뷰

```
당신의 주간 보고서 (2025년 12월 1주)
═════════════════════════════════════════

🔍 핵심 발견
├─ 월-수: 생산성 최고 (평균 7.8/10)
├─ 목-금: 생산성 급락 (평균 5.2/10)
└─ 이유: "주중 스트레스 누적 + 수면 악화"

📊 당신의 강점
├─ 아침형: 아침 7-9시 집중력 최고
├─ 운동 애호가: 하루 1시간 이상 활동 (상위 10%)
└─ 수면 습관: 대부분 날 충분함

⚠️ 개선 영역
├─ 목요일 관리 (프로젝트 재스케줄링?)
├─ 주중 스트레스 완화 (30분 운동 추가)
└─ 주말 회복 부족 (충분하지 않음)

🎯 다음주 목표
├─ 목요일에 스트레스 줄이기 (고강도 일정 피하기)
├─ 저녁 요가 3회 추가
└─ 주말 휴식 시간 +2시간

📈 기대 효과: 평균 생산성 6.5 → 7.1 (↑9%)
```

#### 4.3 월간 깊이 있는 인사이트

```
당신의 12월 깊이 분석 보고서
═════════════════════════════════════════

⚠️ 번아웃 위험 신호
├─ 위험도: 62% (높음)
├─ 신호: Stress ↑38%, Sleep ↓ 1.2시간, Activity ↓25%
├─ 예측: 2주 뒤 번아웃 가능성 높음
└─ 권장: 지금 즉시 개입 필요

🏆 최고 생산성 조건 (당신의 패턴)
├─ 타이밍: 화요일 09:00~11:00
├─ 필요조건: HRV > 40, 수면 7.5시간, 스트레스 < 4
├─ 사전 활동: 아침 운동
└─ 활용: 중요한 업무는 이 시간대에 배치

💡 당신의 개인화된 회복 전략
├─ 스트레스 감소: 운동 (효과도 38%)
├─ 수면 개선: 저녁 요가 (효과도 42%)
├─ 에너지 회복: 오후 산책 (효과도 35%)
└─ 장기 안정성: 주말 완전 휴식 (효과도 52%)

📋 다음달 최적화 계획
└─ 번아웃 방지 + 생산성 최고조 유지
```

### Privacy by Design (기본 원칙)

```

   - 저장: AES-256
   - 전송: TLS 1.2+

   - 언제든 데이터 다운로드 가능
   - 계정 삭제 시 90일 내 완전 제거
   - 제3자와 절대 공유 (동의 제외)

✅ 투명성

   - 권장사항의 근거 명시
```

## 💭 최종 비전

> **"당신의 인생 데이터를 가장 잘 아는 AI"**

모든 사람은:
- 자신의 패턴을 모른다
- 자신에게 뭐가 효과 있는지 모른다
- 미래를 예측할 수 없다
- 어떻게 살아야 하는지 헷갈린다

**Personal Intelligence Platform**은:
- 당신의 모든 데이터를 통합한다
- 당신의 숨겨진 패턴을 찾는다
- 당신에게 뭐가 효과 있는지 알려준다
- 당신의 미래를 예측한다
- 당신이 해야 할 구체적 행동을 제시한다

**결과:**
> 당신이 당신의 삶을 완전히 이해하고, 최적화할 수 있다.
