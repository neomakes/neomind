## 1. 프로젝트 목표
희소한 시계열 건강 데이터를 바탕으로, 개인화된 사용자 활동 트라젝토리를 생성하는 확률적 모델을 개발한다. `Hydra`를 이용한 체계적인 설정 관리와 `W&B`를 통한 실험 모니터링 환경을 구축하여, 효율적이고 재현 가능한 연구 및 개발 파이프라인을 확립한다.

## 2. 핵심 아키텍처
- **개인화된 생성 모델**: VAE(Variational Autoencoder) 아키텍처를 기반으로, 각 사용자의 고유한 특성(`user_id` 임베딩)과 시간의 흐름에 따른 동적 상태 변화를 모두 모델링한다.
  - **VRAE**: 사용자의 '전역적' 특성을 단일 잠재 벡터로 포착하여 분류/클러스터링에 활용.
  - **VRNN**: '매 순간'의 동적 상태를 잠재 변수로 모델링하여 정교한 시계열 생성에 활용.
- **모듈화 및 확장성**: 모델(VRAE, VRNN)과 학습 로직을 분리하고, `Hydra`를 통해 설정을 외부에서 주입하여 유연하고 확장 가능한 구조를 지향한다.

## 3. 프로젝트 상태 및 다음 단계

### 현재 상태
| 항목 | 상태 | 비고 |
|:---|:---|:---|
| **데이터 정제** | ✅ 완료 | `user_id`가 포함된 `.npz` 파일 생성 완료. |
| **모듈형 모델 구현** | ✅ 완료 | `models` 디렉토리 내 `VRAE`, `VRNN` 클래스 구현 완료. |
| **통합 학습 스크립트** | ✅ 완료 | `train.py`에서 Hydra 설정을 통해 두 모델을 선택적으로 학습 가능. |
| **추론 인터페이스** | ✅ 완료 | 각 모델에 자기회귀 방식의 `generate` 메서드 구현 완료. |
| **문서화** | ✅ 완료 | 프로젝트 루트에 핵심 가이드인 `README.md` 생성 완료. |

### 향후 로드맵
- **모델 평가 (`evaluate.py`)**:
  - VRAE의 잠재 벡터 `z`를 이용한 사용자 클러스터링 및 시각화.
  - 생성된 트라젝토리의 통계적 품질(분포 유사성 등) 평가.
- **모델 배포 (`deploy.py`)**:
  - 학습된 모델 가중치를 로드하여, 새로운 사용자에 대한 트라젝토리를 생성하는 독립 스크립트 개발.
  - 초기 컨텍스트를 입력받아 미래 시퀀스를 생성하는 기능 구현.
- **W&B 고급 연동**:
  - 생성된 샘플을 W&B에 시각화하여 로깅.
  - 하이퍼파라미터 스윕 결과 자동 분석 및 리포트 생성.
